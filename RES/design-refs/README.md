# Design reference files

These `DESIGN.md` files were generated by `getdesign` (`npx getdesign@latest add resend` and `add clay`)
and are kept here purely as design inspiration. They are NOT part of the deployed site and are not
linked from `index.html`.

- `resend-DESIGN.md` — primary visual language (editorial-dark developer-tool aesthetic).
- `clay-DESIGN.md` — secondary inspiration (color-cycling cards idea only).

The authoritative visual spec is the "Visual Design Language" section in
`.kiro/specs/advanced-devops-portfolio/design.md`.
